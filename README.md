# Homebrew tap for Clove

Installs the self-contained macOS binary (no Bun).

```bash
brew tap 10ko/clove
brew install clove
```

Via SSH:

```bash
brew tap 10ko/clove git@github.com:10ko/homebrew-clove.git
brew install clove
```

See [github.com/10ko/clove](https://github.com/10ko/clove) for the project.
